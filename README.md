# Chattify

**Chattify** — простое прототипное приложение для обмена сообщениями в реальном времени. Создано в образовательных целях за 2 недели. Поддерживает личные и групповые чаты, несколько типов сообщений и предназначено как демонстрация архитектуры и интеграции с Firebase.

## Краткое описание

Chattify — прототип real-time мессенджера на стеке **React + Next.js + TypeScript** с хранением данных и аутентификацией через **Firebase**. Приложение поддерживает:

-   Личные и групповые чаты
-   Текстовые сообщения
-   Отправку изображений
-   Голосовые сообщения до 30 секунд

Приложение создано как учебный проект, реализовано быстро (≈2 недели), и в нём есть области для улучшения — в первую очередь обработка ошибок и стабильность.

---

## Ключевые возможности

-   Авторизация через Firebase Authentication
-   Список чатов (личных и групповых)
-   Отправка и получение текстовых сообщений в реальном времени (Firestore слушатели)
-   Отправка изображений (Cloudinary)
-   Запись и отправка коротких голосовых сообщений (Cloudinary, до 30 секунд)
-   Состояния загрузки/ошибок (частично реализованы)
-   Управление состоянием темы через Zustand
-   CRUD для сообщений, пользователей, групп

---

## Технологии

-   React
-   Next.js (SSR/SSG возможности, роутинг)
-   TypeScript
-   Zustand (state management)
-   Firebase (Auth, Firestore, Storage)
-   react-firebase-hooks (удобные хуки для интеграции с Firebase)
-   Доп. инструменты: ESLint, Prettier (рекомендации)

## Установка и запуск

1. Клонировать репозиторий

2. Установить зависимости:

```bash
npm install
# или
yarn
```

3. Создать файл `.env` (см. пример ниже) и заполнить переменные Firebase.

4. Запустить в режиме разработки:

```bash
npm run dev
# или
yarn dev
```

5. Построить production-билд:

```bash
npm run build
npm run start
```

## Переменные окружения (пример)

Создайте `.env` в корне проекта и подставьте ваши ключи Firebase:

```
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=messaging_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
NEXT_PUBLIC_FIREBASE_MEASUREMENT_ID=G-XXXXXXX   # опционально
```

## Ограничения и TODO

-   Голосовые сообщения ограничены 30 сек (по требованиям прототипа).

-   Нет продвинутой модерации/фильтрации медиа.
